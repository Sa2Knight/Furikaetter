# Furikaetter -最近のツイートを集計するアレ

* https://furikaetter.herokuapp.com/
* 別途,api-keyを記述したsecret.ymlを配置する必要あり
* Herokuにデプロイする際はsecret.ymlをgitでpushすること

# メモ

* heroku create furikaetter
* git push heroku master
* heroku run bash
* heroku logs --tail
